## Git Commit Convention
| 이모지 | 사용 | 타입 | 상황 |
| --- | --- | --- | --- |
| 🎨 | : art : | design | CSS 등 사용자 UI 디자인 변경 |
| 🥚 | : egg : | style | 코드 포맷 변경, 세미 콜론 누락 |
| ⚡️ | : zap : | feat | 새로운 기능 구현  |
| 📝 | : memo : | docs | 문서 파일 추가 및 수정 |
| ✏️ | : pencil2 : | chore | 단순 오타 수정 |
| 🐛 | : bug : | fix | 버그 수정 |
| ♻️ | : recycle : | refactor | 코드 리팩토링 |
| 🔥 | : fire : | remove | 파일, 코드 삭제 |
| 🚚 | : truck : | rename | 파일, 폴더명, 함수, 변수명 수정 |
| 💡 | : bulb : | comment | 필요한 주석 추가 및 변경 |
